# 🚨 FF03: Modal de envio ao concluir produção

> **Status:** EM ANDAMENTO
> **Criado:** 2026-06-10

## Descrição
Quando um item é movido para "Enviado p/ Entrega" na tela de produção, deve abrir uma modal para registrar informações de envio (transportadora, código de rastreio, valor do frete, observações).
